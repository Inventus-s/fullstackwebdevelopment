This is a Static Position

Static: This is the default position for elements. Elements with a static position will flow normally in the document, according to the order in which they appear in the HTML markup.
This is a Fixed Position

Fixed: Elements with a fixed position are positioned absolutely to the top-left corner of the browser window. This means that the element will stay in the same place on the screen, even when the user scrolls the page.
This is a Relative Position

Relative: Elements with a relative position are positioned relative to their normal position. This means that any top, bottom, left, or right properties that are set will be applied to the element's normal position, rather than the top-left corner of the browser window.
This is a Absolute Position

Absolute: Elements with an absolute position are positioned absolutely to their nearest positioned ancestor. If there is no positioned ancestor, then the element is positioned absolutely to the top-left corner of the browser window.
This is a Sticky Position

Sticky: Elements with a sticky position are positioned relatively until they reach a certain point in the viewport, then they are positioned absolutely. This means that the element will stay in the same place on the screen, relative to the top of the viewport, as the user scrolls.
